Main Features

  • Page Layout:

    The page, has a block structure with division into different sections to better organise the content.

  • Typography:

    Readable fonts are used with well-chosen size and spacing for easy reading.

  • Use of CSS:

    Styles are made using CSS, which allows you to create a modern and attractive page design.

  • Interactive Elements:

    The page has interactive elements such as buttons and controls for the user’s convenience.

  • Cross-browser compatibility:

    The layout of the project is cross-browser compatible to ensure correct display in different web browsers.
